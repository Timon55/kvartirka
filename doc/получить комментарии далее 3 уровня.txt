Для того чтобы получить комментарии к посту до 3 уровня вложенности нужно отправить
get запрос по адресу http://127.0.0.1:8000/api/v1/comment_three_level/{id}/, где параметр id нужно заменить на id конкретного комментария,
например, http://127.0.0.1:8000/api/v1/comment_three_level/4/.
